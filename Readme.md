
# AdventJS Solutions

This repository contains solutions to the [AdventJS](https://adventjs.dev) challenges. AdventJS is an annual event featuring daily JavaScript coding challenges throughout December.

## About

This project showcases solutions to various AdventJS problems, demonstrating problem-solving approaches and JavaScript concepts in action.
